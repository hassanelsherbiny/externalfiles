Medical Lab Reservation System

Application For Managing the Reservation Process And Patients Info For a Medical Lab
Ability To Add / Edit Patient Information And Balance.
Ability To Add / Edit Different Types Of Analysis With Each Price.
Ability To register / Edit an analysis Order Calculating Total Cost And Patient Payments.
Payment Can Be Imediate Or Delayed With Results receipt.
Report For Patients Balance And Delayed Payments.
Report For Analysis Order With Ability To Search By Date Or Get for Current day or month.
Auto BackUp Each 7 Days.
Multiple Application Theme.
Application Language : Arabic
