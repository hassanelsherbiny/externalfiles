Sales Management System

Management System Specified For Small And Midium Sales Store
Storing Products Informations.
Dealing With Multiple Stores With Ability To Add / Remove Store And Change Each Store Info.
Products Can Have Expire Date Or Not With Alert Before It's Expiry Time and Alert For Amount Leak.
Ability To Set Products To Damaged.
Sale System (and Products Return ,Total And Single Sale Bills) With Cash Or Delayed Payment.
Buying System (and Products Return) With Cash Or Delayed Payment.
Add / Edit Delete Customers And Suppliers Information.
Pay / Receive Delayed Payments To / From Customers and Suppliers.
Detailed Report (Daily / Monthly and Between Specific Eras) About Sales / Buying Or Returing Items.
Detailed Report About Supplier Or Customer Transactions And his Dept (Balance).
Detailed Report About Products In Each Store. With Total Cost ,SalePrice And Benefit For each Product And All Products In Whole Store.
Detailed Report (Daily / Monthly and Between Specific Eras) About Safe Tranactions And Ability To Deposite / Withdraw Money From Safe.
Ability To Add Expenses With Expense Reason and Have Detailed Report (Daily / Monthly and Between Specific Eras) About Expenses.
Detailed Report (Daily / Monthly and Between Specific Eras) About Damage Products.
Ability To Set Products Amounts and Prices In Each Store.
Shifts System Allowing Work To Be Seperated Between Different Shifts Storing Start / end Time Of each Shift with Detailed Report After Finishing The Shift.
Adding / Editing / Deleteing Application Users Allowing Each OF The To Do Only Specific Tasks (Users Roles)
Auto BackUp Each 3 Days With Ability To Upload The Back Up To Google Drive
Application Language : Arabic
