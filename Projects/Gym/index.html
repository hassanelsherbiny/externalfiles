Gym Management System

System For Managing a Gym Business and Members Attendance
Ability To add Multiple Membership Types And Changing Each Type Specification
Ability To Add / Edit Member Information And Take photo Of Him/here
Ability To Add / Renew Each Member's Subscription
Storing the Number and the duration Of Each Member Attendance
Hold And Resume Member Subscription with calculating the holding period
Adding / Detailed Report Of Expenses
Adding / Editing / Deleteing Application Users Allowing Each OF The To Do Only Specific Tasks
Storing Information Of The Daily Member (visitor).
Report of Newly Added Or Renewed Subscriptions With Ability To Search Between 2 specific Eras
Auto BackUp Each 3 Days With Ability To Upload The Back Up To Google Drive
Multiple Application Theme.
Application Language : Arabic
