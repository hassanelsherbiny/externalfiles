Open Book

Online E-Book Library And BookStore
Open Book is a WebSite For Displaying Books Organized By It's Category Or It's Authors.
The Site Is Used As E-Commerece Site For Selling Books And Adding Information About The Store.
Site Has Two Actors : The Admin And The User.
Login Or Register New Account.
User Can Browse All Aviable Books And Order Any Of them If it Has Paper Version.
User Can Search for All Aviable Books ,Categories Or Authors By Containg Letters.
Can Also Request Books Which Doesn't Exist On The Site.
Admin : Add / Edit / Delete Authors , Categories ,Books And Download Links For each Book And Prices For Paperbook version.
View All Requests And Orders.
Orders And Requests Are Delivered To Admin Giving an Imediate Alert (With help Of SignalR And JQuery).
Books, authors ,Categories , Store Information Are All Organized And built According To Special Schemas Of Schema.Org
